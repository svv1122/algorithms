Algorithms
==========

This project aims to provide a detailed introduction to as many algorithms as possible.
Each algorithms is implemented in **Rust**, with a focus on clarity and completeness.

Note: This repository includes **only the implementation code**. Test cases, benchmarks, and usage examples are **not** included at this stage.

Project Goals
-------------

- Learn and implement algorithms as completely as possible
- Understand how they are implemented in Rust
- Explore more complete and formal definitions
